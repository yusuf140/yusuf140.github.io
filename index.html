Penjelasan Tag HTML

Penjelasan Tag HTML

Di bawah ini adalah penjelasan tentang beberapa tag HTML dalam kode yang diberikan:

  • <!DOCTYPE html>: Deklarasi tipe dokumen HTML5.
  • <html lang="en">: Elemen root dokumen HTML dengan atribut lang yang menunjukkan bahasa.
  • <head>: Bagian kepala dokumen yang berisi metadata.
  • <meta charset="UTF-8">: Menentukan encoding karakter dokumen.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Pengaturan tampilan responsif.
  • <title>: Menentukan judul halaman.
  • <style>: Aturan CSS untuk mengatur tampilan.
  • body: Mengatur properti CSS pada elemen body.
  • .container: Mengatur properti CSS pada elemen dengan class container.
  • h1, h2: Mengatur properti CSS pada heading level 1 dan 2.
  • p: Mengatur properti CSS pada elemen paragraf.
  • ul: Mengatur properti CSS pada elemen unordered list.
